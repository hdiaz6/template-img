# template-img

## Descripción

Este repositorio está diseñado para **almacenar y gestionar imágenes**. Aquí se encuentran todas las imágenes necesarias para los proyectos, plantillas y documentación.

## Estructura

El repositorio contiene las siguientes carpetas principales:

- **BBOG-COB-NOTIFICATION-NEG-MAIL/**: Imágenes relacionadas con notificaciones de correo electrónico

## Uso

Para utilizar las imágenes de este repositorio:

1. Clona el repositorio
2. Accede a la carpeta correspondiente según el tipo de imagen que necesites
3. Descarga o referencia la imagen en tu proyecto

## Contribución

Si deseas añadir nuevas imágenes:

1. Organiza las imágenes en la carpeta aproppiada
2. Asegúrate de que los nombres de archivo sean descriptivos
3. Realiza un commit con una descripción clara

## Formatos soportados

Se aceptan imágenes en los siguientes formatos:
- PNG
- JPG/JPEG
- GIF
- SVG
- WebP

---

**Nota:** Este repositorio está destinado exclusivamente al almacenamiento de imágenes.
